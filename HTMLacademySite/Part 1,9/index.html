Part 1.9. Формы.

<form>
Форма - это тэг, помогающий отправить информацию с веб-страницы на веб-сервис.
У этого тэга есть атрибуты - action и method.
action - нужен для того, чтобы задать ссылку для отправки информации (куда).
method - нужен для того, чтобы передать метод отправки информации.
<form action="#" method="get"></form>
<input>
Инпут - это тег, который создает поле для ввода данных. Это может быть текст (в онсновном его используют).
У него есть атрибуты type и name.
type - нужен для того, чтобы задавать тип полю ввода.
name - нужен для того, чтобы задавать имя полю (зачем оно).
