Part 1.9. Формы.

HRIPNV

<form>
Форма - это тэг, помогающий отправить информацию с веб-страницы на веб-сервис.
У этого тэга есть атрибуты - action и method.
action - нужен для того, чтобы задать ссылку для отправки информации (куда).
method - нужен для того, чтобы передать метод отправки информации.
<form action="#" method="get"></form>
<input>
Инпут - это тег, который создает поле для ввода данных. Это может быть текст (в онсновном его используют).
У него есть атрибуты type и name.
type - нужен для того, чтобы задавать тип полю ввода.
name - нужен для того, чтобы задавать имя полю (зачем оно).
Атрибут id
Этот атрибут служит для персональной идентификации элемента HTML.
Есть также атрибут value, который определяет значение поля ввода.
Идеальный пример: сайт Госуслуги. Большая анкета (любая). Нужно все заполнить. Но вместо того, чтобы искать, как это все заполнять, в строках уже написан текст того, что нужно писать.

Что такое <input>?

<input> - это тег, который создает поле ввода. Оно собирает информацию от пользователей.
Имеет следующие атрибуты : type, id, name, value.

type - это атрибут, который задает занчение того, как будет выглядеть поле ввода. Оно может быть для текста, для пароля, для отметок(чекбокс).
Можно сделать кнопу при помощи значения submit.

Что такое <label>?

<label> - это тег, который связывает <input> и текст логически.
Имеет атрибут for, который помогает обратиться <label> к <input>, путем ввода в for идентификатора поля воода.
